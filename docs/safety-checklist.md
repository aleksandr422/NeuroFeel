# Safety Release Checklist

- Crisis lexicon review date is under 180 days.
- Resource phone numbers/URLs are verified within 90 days.
- Scope notices are visible on onboarding, summary cards, and first entry-sheet open per session.
- Marketing copy contains none of banned overpromise phrases.
- PIN flow requires explicit loss-recovery acknowledgement before save.
- Destructive actions still require confirmation gate.
- Crisis match path verified: entry saves, support card shows, no auto-tags/auto-observations, no telemetry.
